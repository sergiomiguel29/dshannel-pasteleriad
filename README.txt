D'Shannel Pastelería - Frontend

Archivos principales:
- index.html
- style.css
- script.js
- assets/logo.jpeg
- assets/hero-pasteleria.png

Funciones incluidas:
- Página profesional con portada, servicios, formulario de contáctenos y redes sociales.
- Constructor de tortas con moldes explicados en centímetros y porciones.
- Monto aproximado solo dentro del constructor personalizado.
- Servicios sin precios públicos, con apartado "Qué incluye".
- Enlaces reales a Facebook y TikTok.

Para probar:
Demo GitHub Pages:
- Esta versión sirve para que el cliente vea la página desde un link.
- GitHub Pages no ejecuta PHP, por eso la subida real de foto como enlace funciona solo en hosting con PHP.
- WhatsApp, diseño, cotizador, redes y páginas legales sí funcionan como demo.
